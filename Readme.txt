Projeto livre para a atividade 2 da matéria de Engenharia de Software.

HUB de funções divertidas e não muito uteis com o objetivo de exemplificar a utilização do git flow em um projeto

Funcionalidades:
    1-feature: A primeira funcionalidade adicionada é uma função recursiva que printa os numeros impares positivos até o N definido pelo usuário;

    3-feature: A terceira funcionalidade adicionada foi uma receitinha de pao de queijo que amo, recomendo muito;
